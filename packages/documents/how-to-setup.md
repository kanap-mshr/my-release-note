
# セットアップ手順メモ

---

## 初期セットアップ

### Hugoインストール

```bash
brew install hugo
```

### サイトの雛形作成

```bash
hugo new site my-release-note

cd my-release-note
git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c
vim config.toml
```
